Запросы

GET - для чего: получить данные
Safe: да
Идемпотентность: да, повтор даёт один и тот же результат
Кэшируемость: да
Не использовать: для создания или изменения данных


POST - для чего: что-бы создать новый ресурс
Тело запроса: JSON
Safe: нет
Идемпотентность: нет, повтор запроса создаст дупликат объекта
Кэшируемость: нет
Не использовать: для того, чтобы получить данные


PUT - для чего: полная замена ресурса
Тело запроса: JSON
Safe: нет
Идемпотентность: да
Кэшируемость: нет
Не использовать: если меняется только часть полей или нет возможности отправить объект целиком


PATCH - для чего: частичное обновление
Тело запроса: JSON
Safe: нет
Идемпотентность: да
Кэшируемость: нет


DELETE - для чего: удаление ресурса
Тело запроса: нет
Safe: нет
Идемпотентность: да
Кэшируемость: нет
Не использовать: если запрещено физическое удаление данных


HTTP - статусы

200 ОК успешное чтение или обновление ресурса
201 Created успешное создание ресурса (POST)
204 No Content успешное обновление без тела

400 Bad Request не валидные данные (валидация DTO - Data transfer object)
404 Not Found ресурс не найден
409 Conflict конфликт состояний
500 Internal Server Error - неожиданная ошибка

GET: /api/cars выдаёт список автомобилей
GET: /api/cars/11 выдаёт авто по ID
